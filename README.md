# propagator-and-woodbury-identity

このリポジトリは、Weinberg の量子場理論における式 (21.2.14) から式 (21.2.21) への導出を、Woodbury の行列恒等式を使って段階的に整理したメモです。

## 目的

- スカラー場の伝播関数を、行列演算の流れが追える形で再導出する
- Goldstone 方向の性質を、計算上どこで使っているか明示する
- Woodbury 恒等式を使う必然性を、物理的な文脈とあわせて理解する

## 主な内容

- 二次ラグランジアンから運動量空間の作用素を抽出
- 質量行列の分解と性質の確認
- Woodbury 恒等式の適用による逆行列の評価
- 最終的な伝播関数の成分表示への書き下し
- この手法が他分野でも使われる代表例の整理

## 読み方の推奨順

1. まず導出対象の式 (21.2.14) と (21.2.21) を確認する
2. 質量行列の分解と条件を把握する
3. Woodbury 恒等式の代入対応を追う
4. 最終式と物理的意味を照合する

## 補足

- ノート内では、教科書の係数に関する注意点にも触れています。
- 目的は厳密な再現計算の補助であり、講義用の最短メモではなく、途中式を省略しない解説に寄せています。
